# Pattern Generator and Number Analyzer

A simple Python console application that allows users to:

- Generate a star (`*`) pattern.
- Analyze a range of numbers.
- Identify whether each number is even or odd.
- Calculate the sum of numbers within the given range.

## Features

- Interactive menu-driven program
- Star pattern generation
- Even/Odd number analysis
- Sum calculation
- Simple and beginner-friendly Python project

## Requirements

- Python 3.x

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/yourusername/pattern-number-analyzer.git
```

2. Navigate to the project folder:

```bash
cd pattern-number-analyzer
```

3. Run the program:

```bash
python project2.py
```
